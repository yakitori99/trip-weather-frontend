## trip-weather-frontend
trip-weatherのfrontend側ソースコードです。

### 主な利用技術
メイン
- Vue.js
- vuetify

グラフ表示
- chart.js + vue-chartjs

その他
- Font Awesome


### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```
